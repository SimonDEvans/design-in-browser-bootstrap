Thanks for installing the Design In the Browser Bootstrap

Before beginning work, the following are required:

  1. The project must be renamed in package.json.

    Change the line that reads "name": "DesignInTheBrowserBootstrap", to specify your new project's name e.g "name": "Client XYZ",.

  2. A fresh git repository must be created:

    In the same folder as the file package.json, run:
    rm -rf .git && git init && git add . && git commit -m "First commit"

    This new repository will need to be uploaded somewhere eventually e.g Github, but this can be left till later.

  3. Create a staging-ssh-config.js file by renaming staging-ssh-config.example.js. See the README.md "Deploying to a staging server"->"Setup per project" for information on how to configure it.

  4. Delete all the HTML inside {% raw %}{% block content %}{% endblock %}{% endraw %} block in this file, and start writing your own!
